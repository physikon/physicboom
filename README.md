# Учебный сайт 9 класс

## Описание
Это учебный сайт для школьников 9 класса с формулами по физике, тестами и другими предметами.

## Технологии
- HTML
- CSS
- JavaScript

## Как использовать
1. Клонировать репозиторий или скачать файлы
2. Открыть index.html в браузере
3. Выбирать предметы через меню
